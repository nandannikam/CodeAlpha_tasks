## 🛠️ Projects Included
**1. Basic Banking System**
A comprehensive console-based banking application simulating real-world bank operations.

  * Key Features: Secure account creation, PIN-protected login, deposits, withdrawals with limit checks, inter-account fund transfers,                   and chronological transaction history.

  * Core Concepts: Object-Oriented Programming (Inheritance, Encapsulation), File I/O (<fstream>), Standard Template Library                              (std::vector).

**2. GPA & CGPA Calculator**
An interactive academic tool designed to calculate a student's Grade Point Average (GPA) for individual semesters and their Cumulative Grade Point Average (CGPA) over time.

 * Key Features: Dynamic course and credit input, standardized 4.0 grading scale mapping, and robust input validation for letter                        grades.

  * Core Concepts: Procedural Programming, Control Flow (loops, conditionals), Type Casting, and Mathematical Logic.
